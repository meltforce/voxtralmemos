## Description (max 4000 characters)

Record a voice memo. Get an instant transcription and a smart summary. It's that simple.

Voxtral Memos uses Mistral AI's Voxtral speech model to turn your recordings into accurate, readable text — and then summarizes, extracts action items, or transforms them however you like.

TRANSCRIBE ANYTHING
Record meetings, lectures, ideas, or quick thoughts. Voxtral Memos transcribes them instantly using Mistral AI's state-of-the-art Voxtral model. Supports 13+ languages including English, German, French, Spanish, and more.

SMART SUMMARIES & CUSTOM PROMPTS
Go beyond transcription. Get concise summaries, extract to-do lists, create journal entries, or define your own custom prompt templates. Every memo can be transformed in multiple ways.

BUILT ON EUROPEAN AI
Powered by Mistral AI, Europe's leading AI company based in Paris. Your data is processed by a GDPR-compliant, European technology provider.

PRIVACY BY DESIGN
- Powered by Mistral AI — a European, GDPR-compliant AI provider
- No account required — ever
- No tracking, no analytics, no telemetry
- Zero third-party SDKs
- All data stored locally on your device
- Your API key is secured in the iOS Keychain
- Audio is only sent when you explicitly request transcription
- Fully open source on GitHub

BRING YOUR OWN KEY
Voxtral Memos uses a Bring Your Own Key model. You connect your own Mistral API key, so you're always in control of your usage and costs. No subscriptions, no hidden fees.

FREE & OPEN SOURCE
The app is completely free. No ads, no paywalls, no subscriptions. The full source code is available on GitHub under the MIT license. If you enjoy the app, you can leave a tip to support development.

REQUIREMENTS
- A Mistral AI API key (free to create at console.mistral.ai)
- iOS 26.0 or later
